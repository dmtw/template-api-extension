# template-api-extension

> Scaffold out a Colmena API Extension

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
sao colmena/template-api-extension my-extension
```
